# Results

This folder contains selected final results used by the poster, report, and README.

- `metrics_table.csv`: summary table of the main MSE results.
- `forecasting_gain_user.png`: forecasting gain over the PatchTST/64 paper baseline.
- `scale_sensitivity_user.png`: temporal coarse-graining / scale-sensitivity plot.

The notebook may generate additional curves and checkpoints under `experiment_artifacts/`; those files are not committed by default because they are machine-specific and can be large.
